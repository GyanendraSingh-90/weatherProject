**Weather Forecast Web App**

**Course assignment**=> Weather forecast application by using HTML, JavaScript, and Tailwind CSS.

**Project Overview**
Weatherly fetches weather data from OpenWeatherMap and displays:-
1. Current weather for a searched city or your current location.
2. 5-day forecast (aggregated from 3-hour forecast).
3. Recent searches dropdown stored in local storage.
4. Celsius â†” Fahrenheit toggle for today's temperature (only).
5. Custom alerts for extreme temperatures (heat/cold).
6. Dynamic background based on weather (rainy/sunny/cloudy).
7. Error handling with friendly UI popups.

**Features**
1. City search + Enter key support
2. Geolocation search (browser permission required)
3. Recent cities dropdown (localStorage)
4. Error messages rendered on UI (no alert())
5. Responsive layout (desktop, tablet, phone)
6. Icons from OpenWeatherMap
